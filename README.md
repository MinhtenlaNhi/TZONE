# Online Course Platform

Fullstack starter for an online course selling website.

## Project Structure

- `client`: React frontend (Vite)
- `server`: Node.js + Express backend
- `database`: DB connection and seed data
- `docs`: API docs and ERD notes

## Quick Start

```bash
npm run install:all
npm run dev
```

Default ports:
- Frontend: `5173`
- Backend: `5000`
